# Decision log (ADR)

Спільний журнал затверджених рішень партнерства. **Джерело правди для всіх
Claude-сесій обох партнерів** — Claude Code зобовʼязаний дотримуватись рішень
із цієї теки (правило в CLAUDE.md).

## Як це працює

1. Ідея/питання обговорюється де завгодно (дзвінок, чат, GitHub Discussions,
   колонка Ideas на дошці).
2. Коли рішення дозріло — оформлюється файлом у цій теці (копіюй `TEMPLATE.md`,
   наступний номер: `NNNN-коротка-назва.md`) і відкривається PR.
3. **Рішення вважається затвердженим, коли PR змерджив ІНШИЙ партнер**
   (не автор). Merge = підпис.
4. Рішення не редагуються заднім числом: якщо щось змінилось — новий ADR, що
   позначає старий як superseded.

## Статуси

- `proposed` — PR відкрито, чекає на другого партнера
- `accepted` — PR змерджено
- `superseded by NNNN` — замінено новішим рішенням
